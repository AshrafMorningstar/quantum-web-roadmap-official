# Quantum Git Visualization Platform

**Author**: Ashraf Morningstar  
**GitHub**: https://github.com/AshrafMorningstar

A visualization tool that represents Git history as a graph of quantum states.

## Features

- **D3.js Graph**: Interactive force-directed graph of commits
- **Interactive Nodes**: Click nodes to view commit details "collapsed" from the probability cloud
- **Branch Visualization**: Color-coded branches representing different timelines

## Usage

Drag nodes to rearrange the history. Click on any node to inspect its detailed "quantum state" (commit info).
