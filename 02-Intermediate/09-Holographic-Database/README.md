# Holographic Database Interface

**Author**: Ashraf Morningstar  
**GitHub**: https://github.com/AshrafMorningstar

A 3D interface for database management where data nodes are represented as physical objects in a holographic space.

## Features

- **3D Node Visualization**: Data points rendered as interactive icosahedrons in 3D space.
- **Raycasting Interaction**: Hover and click on nodes to retrieve their metadata.
- **Connection Graph**: Visual representation of relationships between data points.

## Usage

Rotate the view by dragging (or wait for auto-rotate). Click any node to inspect its properties in the HUD. Use the command bar to simulate queries.
