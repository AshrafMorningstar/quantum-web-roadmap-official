# Temporal Web Browser

**Author**: Ashraf Morningstar  
**GitHub**: https://github.com/AshrafMorningstar

A concept browser that allows users to surf not just across URLs, but across _time_.

## Features

- **Temporal Slider**: Scrub from the 1990s to the 2030s.
- **Dynamic Era Rendering**: The browser engine automatically switches rendering engines (CSS/HTML structure) to match the selected historical or future era.
  - **1990s**: Times New Roman, grey backgrounds, basic HTML.
  - **2000s**: Gradients, Verdana, "Web 2.0" aesthetics.
  - **2020s**: Clean, flat design, responsive grids.
  - **2030s**: Dark mode, neon, holographic/AR interfaces.

## Usage

Drag the bottom slider to travel through time and watch the webpage rewrite itself to match the design language of that era.
