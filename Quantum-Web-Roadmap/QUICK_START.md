## üêõ TROUBLESHOOTING

**Project won't load?**

- Ensure you're opening the HTML file in a browser
- Check browser console for errors (F12)
- Verify internet connection (for CDN libraries)

**Webcam not working? (P01)**

- Grant camera permissions
- The project falls back to simulation mode automatically

**GitHub API rate limited? (P02)**

- Wait 60 minutes or use a different username
- The demo works with any valid GitHub user
